JavaScript Tutorial

Functions

A function in JavaScript is a code element that makes something happen when it is called. To make a function, first type function, then create a name for your function, and follow it with (). Any needed parameters go into the (). A parameter is a variable that is used when declaring a function to define the scope of that function's activity. Follow the parameter with {} that contain the information about what you want the function to do. Functions allow you to run the same action multiple times without having to retype it each time you need it. An argument is a variable that provides the data to determine the value of the function. You can change the arguments used to get different results from one function.

Objects

An object is a variable that can contain many values. These values are written as name:value and are called properties. Like any real-life object, JavaScript object properties define the characteristics of that object. For example, a house has walls, ceilings, floors, paint colors, etc. Object names and property names are case sensitive, and their definitions can span multiple lines. Objects can be initialized using the literal notation (initializer notation). An object initializer is a list of zero or more pairs of property names and associated values of an object, separated by commas and enclosed in curly braces ({}).

Arrays

An array is a single variable that is used to store different elements. Numbers, Strings and Boolean can be stored in a single array. Arrays in JavaScript are indexed from 0 so we can access array elements as follows: the first element of an array is at index 0, and the last element is at the index equal to the value of the array's length property minus 1. Array objects are iterable and can loop elements through for...of loops.
